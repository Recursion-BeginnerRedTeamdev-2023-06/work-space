# Red Team 成果物：テトリス
※https://recursion-beginnerredteamdev-2023-06.github.io/work-space/ をクリックすると利用可能です。

## <テトリスゲームの特徴>

- スコアが上がるとゲームスピードが上がり、難易度が上がります。
- 4000 点以降からゲームスピードは最大になりあがりません。
- テトリスブロックの消える段が増えるとボーナスが追加されて高いスコアが追加されます。

## <テトリスゲームの説明>

- 本ゲームのボタンについて説明を記載してます。
- テトリミノの操作はキーボード操作します。操作説明は index.html に直接記載されてます。

### 1. GAME を開始する。

1. 左側の GAME START と記載されている青色ボタンをクリックする。<br>
   ![initGame](https://github.com/teamdev-test-red/work-space/assets/72297791/d2710bcd-2c7b-4240-8020-4869a32e42a8)
1. GAME が開始します。
   ![gameStart](https://github.com/teamdev-test-red/work-space/assets/72297791/d014e9f3-f73f-448c-bec8-b87d0d54b63b)
   )

### 2. GAME を一時停止する。

1. 左側の GAME STOP と記載されている緑色ボタンをクリックする。<br>
   <img width="1108" alt="スクリーンショット 2023-06-30 220301" src="https://github.com/teamdev-test-red/work-space/assets/135023031/17745dd0-b078-4c70-a40f-c9df30231b09">

1. GAME が停まる。<br>
   <img width="1082" alt="スクリーンショット 2023-06-30 220405" src="https://github.com/teamdev-test-red/work-space/assets/135023031/7b95f48d-6264-4caf-8c21-b73f18a1839d">

### 3. GAME を一時停止から再開する。

1. 左側の GAME RESTART と記載されている緑色ボタンをクリックする。<br>
   <img width="1082" alt="スクリーンショット 2023-06-30 220405" src="https://github.com/teamdev-test-red/work-space/assets/135023031/7b95f48d-6264-4caf-8c21-b73f18a1839d">

1. STOP 状態が解除され、GAME が再開する。
   <img width="1082" alt="スクリーンショット 2023-06-30 220405" src="https://github.com/teamdev-test-red/work-space/assets/135023031/7b95f48d-6264-4caf-8c21-b73f18a1839d">

### 4. GAME Over 後、再挑戦する。

1. 左側の GAME RETRY と記載されている青色ボタンをクリックする。<br>
   <img width="1082" alt="スクリーンショット 2023-06-30 220826" src="https://github.com/teamdev-test-red/work-space/assets/135023031/5ba8c3e9-2a9d-4b70-8bd9-4848d0e3c77f">

1. スコアが初期化された状態になる。青色の GAME START ボタンをクリックすると開始する。
   ![initGame](https://github.com/teamdev-test-red/work-space/assets/72297791/d2710bcd-2c7b-4240-8020-4869a32e42a8)
